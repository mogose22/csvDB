Есть база данных номеров представленная в виде csv-файла: Петя;89225225522
необходимо написать программу, которая позволяет:
1) осуществлять поиск по имени, номеру (поиск в словаре с повторяющимися ключами либо поиск по значению ключей)
2) добавлять и удалять записи (корректная обработка повторов, если заморочиться- то повторов номеров быть не должно. Номер- ключ?)
база данных должна хранить записи в отсортированном виде (просто используем multimap и по именам автоматически сортируется)
Примечание 1. По максимуму используйте потоки (забудьте про всякие fopen и т.д.)
Примечание 2. По максимуму используйте STL. Обратите внимание, что имена могут дублироваться.
