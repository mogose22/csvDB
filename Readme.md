Есть база данных номеров представленная в виде csv-файла:
	Антон;неизвестно
	Николай;89225270301
	Юра;неизвестно
необходимо написать программу, которая позволяет:
1) осуществлять поиск по имени, номеру
2) добавлять и удалять записи
база данных должна хранить записи в отсортированном виде
Примечание 1. По максимуму используйте потоки (забудьте про всякие fopen и т.д.)
Примечание 2. По максимуму используйте STL. Обратите внимание, что имена могут дублироваться.
